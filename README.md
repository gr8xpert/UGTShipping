# AL UROOJ - General Trade & Shipping

A modern, responsive landing page for AL UROOJ General Trade & Shipping LLC, a maritime services company based in Salalah, Oman.

## Overview

AL UROOJ provides comprehensive shipping and trade services across all major Omani ports including Salalah, Duqm, Muscat, and Sohar.

## Services

- **Commodities Trading** - Import/export of various commodities
- **Animal Feed & Fodder Trading** - Livestock feed supply chain
- **Ship Handling & Provisions** - Port services and vessel supplies
- **Vessel Sale & Purchase** - Maritime brokerage services
- **Demolition Brokerage** - Ship recycling facilitation
- **Shipping Consultancy** - Expert maritime advisory

## Tech Stack

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, backdrop-filter)
- Vanilla JavaScript
- Responsive design (mobile-first approach)

## Features

- Responsive navigation with mobile menu
- Parallax hero section
- Service cards with hover effects
- Port locations showcase
- Contact form
- SEO optimized with Schema.org structured data
- Social media meta tags (Open Graph, Twitter Cards)
